# Agentic Notepad

Este projeto demonstra um agente de IA simples capaz de gerenciar um diário. Ele utiliza o framework **LangGraph** juntamente com **Node.js**.

## Requisitos

- Node.js 20+
- Uma chave de API da OpenAI disponível na variável `OPENAI_API_KEY`.

## Como executar

Instale as dependências:

```bash
npm install
```

Em seguida execute o agente:

```bash
npm start
```

O agente criará e atualizará um diário chamado `journal` dentro da pasta `diaries/`.
